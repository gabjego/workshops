Mr. G workshops - Landing Page de Formación y Consultoría

Esta es una aplicación web de una sola página (SPA) diseñada para transformar la marca personal de Mr. G en una plataforma transaccional de venta de talleres y servicios de consultoría empresarial. El sitio destaca por un diseño moderno de alta gama, optimizado para la conversión y totalmente adaptado a dispositivos móviles.

🚀 Propuesta de Valor

El sitio vincula la irreverencia y el pensamiento crítico de las redes sociales con una trayectoria docente de más de 20 años, utilizando el humor como vehículo para el anclaje de conocimiento.

🛠️ Tecnologías Utilizadas

HTML5 & JavaScript (Vanilla): Lógica de navegación y manipulación dinámica del DOM.

Tailwind CSS: Framework de estilos utilizado para el diseño responsivo y la estética "Executive Cyan".

Chart.js: Visualización de la "Ecuación de Resultados" y otros elementos analíticos.

Google Fonts: Tipografías especializadas (Inter y Lexend) para una legibilidad superior.

✨ Características Principales

Arquitectura de Información: Separación clara entre pilares pedagógicos y oferta comercial.

Catálogo Interactivo: Sistema de filtrado por categorías (Padres, Comunicación, Idiomas, Redes, Casos de Estudio).

Casos de Estudio Empresariales: Análisis profundo de modelos de negocio reales (Netflix, Kodak, Costco, etc.) para validar la autoridad del consultor.

Formulario de Inscripción Inteligente: Sistema que pre-redacta la solicitud de información basándose en el taller seleccionado por el usuario.

Optimización Móvil: Diseño adaptativo que prioriza la visibilidad de la marca y la facilidad de navegación en smartphones.

📁 Estructura del Contenido

Hero Section: Introducción con imagen de marca personal y llamada a la acción principal.

Metodología: Explicación técnica de la "Ecuación de Resultados" (Humor + Rigor Académico).

Catálogo: Galería dinámica de talleres y estudios de caso.

Contacto: Formulario directo integrado para captación de leads.

🌐 Publicación y Despliegue

Para poner este sitio en línea, se sugieren las siguientes opciones:

Netlify Drop (Más rápido): Arrastra el archivo mister_g_vende.html a Netlify Drop.

GitHub Pages: Sube el archivo a un repositorio y activa la opción de Pages en la configuración.

Wix Integration: Utiliza el widget "Incrustar HTML" en tu editor de Wix y pega el código completo.
